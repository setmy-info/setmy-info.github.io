# Laminar

## Information

`Laminar` is a platform focused on observability, tracing, and evaluation for LLM applications and agents.

It helps teams understand how prompts, tools, and multi-step AI workflows behave in development and production.

### What is it for?

Typical `Laminar` use cases include:

* tracing LLM and agent workflows
* debugging prompt and tool execution paths
* measuring latency, token usage, and quality signals
* running evaluations for model and prompt changes
* improving visibility into production AI systems

## Usage

`Laminar` is typically used together with application frameworks rather than instead of them. It fits teams that want
observability and evaluation on top of existing AI integrations.

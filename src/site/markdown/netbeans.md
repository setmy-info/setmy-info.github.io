## Usage, tips and tricks

### Font size

```sh
nano /opt/netbeans/etc/netbeans.conf
```

```
netbeans_default_options="...  --fontsize 12"
```

### Temp folder

To change home folder (that can be too small)

```sh
nano /opt/netbeans/etc/netbeans.conf
```

```
netbeans_default_options="...  -J-Djava.io.tmpdir=/tmp/netbeans"
```

# Yi

## Information

`Yi` from `01.AI` is an open-license model family with good multilingual support. It is useful when you want a more
open deployment option and care about language coverage beyond English-only workloads.

## Common use cases

* multilingual chat and assistant workflows,
* open-model experimentation,
* and deployments where licensing openness matters.

## Practical note for Ollama users

Confirm the latest `Yi` model tags and packaging details in `Ollama` before building workflows around a specific local
variant.

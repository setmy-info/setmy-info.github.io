## Information

## Installation

Install Hugo CLI.

## Usage, tips and tricks

```
hugo version
hugo new site hugo-test-blog
cd hugo-test-blog
echo git init
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
echo "theme = 'ananke'" >> hugo.toml
hugo server
firefox --new-tab http://localhost:1313/
```

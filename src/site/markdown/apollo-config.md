# Apollo Config

## Information

`Apollo Config` is an open-source centralized configuration management platform.

It is used to manage application configuration across environments such as development, testing, and production, with
features such as dynamic updates, versioning, namespace-based organization, and change tracking.

### What is it for?

Typical `Apollo Config` use cases include:

* centralized application configuration management
* environment-specific settings for `dev`, `test`, and `prod`
* dynamic configuration refresh without full redeployment
* configuration versioning and audit history
* governance for shared and application-specific settings

## Usage

`Apollo Config` is commonly used in `Java` and `Spring` ecosystems, especially in larger internal platforms where teams
want a dedicated configuration service instead of embedding all runtime configuration into deployment files or service
discovery tools.

## Usage, tips and tricks

* Split the track Ctr + I
* Ctrl + x Track → Add new → Stereo ... mono
* In the panel selection tool I mark start with Shift + Click on many tracks, mark the selection end with a selection
  tool, movable is highlighted.
* Amplify: Effects → Volume and Compression → Amplify

### Small garage echo

* Effect → Delay and reverb
* Room Size: 25–40%
* Reverberance: 20–35%
* Damping: 40–60%
* Wet Gain: –6 kuni –10 dB
* Dry Gain: 0 dB
* Stereo Width: 60–80%
* Pre-delay: 10–20 ms

### Small echo room

Volume +  Reverb + EQ + Delay

1) Volume: Effect -> Volume and Compression -> Amplify

-10 dB (Can't be a lot reduced, because need to hear the original)

2) Reverb: Effect → Delay and reverb

| Seade        | Väärtus  | Miks                                      |
|--------------|----------|-------------------------------------------|
| Room Size    | 20–30%   | Väike tuba, mitte suur saal               |
| Reverberance | 15–25%   | Väike kaja, ei domineeri                  |
| Damping      | 50–60%   | Kõrged sagedused summutatud → heli tuhmem |
| Wet Gain     | –10 dB   | Kajaga heli ei ole liiga vali             |
| Dry Gain     | 0 dB     | Originaal hääl jääb alles                 |
| Stereo Width | 40–60%   | Väike ruum, õrn ruumiline tunne           |
| Pre-delay    | 10–20 ms | Natuke aega, enne kui kajad algavad       |

3) EQ: Effect → EQ and Filters → Filter Curve EQ

100Hz 2dB , 5000Hz 10dB -2dB, 20000Hz 20dB -8dB

4) Delay: Effects -> Delay and reverb -> Delay

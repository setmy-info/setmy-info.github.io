# LM Studio

## Information

`LM Studio` is a desktop application with a graphical interface for downloading, running, and chatting with local
language models.

It is often used by people who want a simple GUI instead of command-line based local model tooling.

### What is it for?

Typical `LM Studio` use cases include:

* testing local models through a simple desktop GUI
* comparing prompts and model behavior interactively
* exposing a local API for nearby applications or experiments
* trying quantized models without setting up a full server stack
* demonstrating local AI capabilities to non-technical users

## Usage

`LM Studio` is especially useful for local experimentation, demos, and small personal workflows where ease of use is
more important than production-scale serving features.

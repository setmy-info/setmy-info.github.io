# Haystack

## Information

`Haystack` is an open-source framework for building search, question answering, and retrieval-augmented generation
applications.

It is well known for combining document pipelines, retrievers, generators, and evaluation tooling in one framework.

### What is it for?

Typical `Haystack` use cases include:

* enterprise search
* document retrieval and question answering
* RAG pipelines with reranking and filtering
* indexing and querying large document collections
* evaluation of retrieval and generation quality

## Usage

`Haystack` is a strong choice when search and retrieval are central to the application architecture and you want a
pipeline-oriented approach.

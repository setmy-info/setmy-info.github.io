# Car workstation project

Monitor, keyboard, mouse, desk, dock, laptop, 5G internet place. Back seat, desk leans on front seat bus. Another
leaning point at the door side and one more at the middle in the car trash can (removed).

Desk max length 1000mm. Seating place length 540 (from dor to middle). Table bust be reversible, so you can choose which
side to work on. Constructed using [T-Slot frames](buildingblocks.md#t-slot-frames).

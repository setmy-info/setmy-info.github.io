# T-slot

## Information

T-slot aluminum extrusion profiles are structural aluminum sections with T-shaped grooves running along their length.
The slots accept T-nuts, which allow bolts and brackets to be attached at any position along the profile without
drilling. This makes them highly versatile for machine building, CNC router frames, workbenches, 3D printer frames,
industrial automation jigs, and custom enclosures.

**Common profile sizes (width x height in mm):**

| Size      | Typical use                                      |
|-----------|--------------------------------------------------|
| 20 x 20   | Light-duty frames, 3D printer structures         |
| 30 x 30   | Medium-duty frames                               |
| 40 x 40   | Heavier CNC and machine frames                   |
| 45 x 45   | Industrial-grade structures, workbenches         |
| 80 x 80   | Very heavy-duty machine bases                    |

**Key components:**

* **Profile / extrusion** — the aluminum bar itself
* **T-nuts** — slide into the slot and accept M5/M6/M8 bolts
* **Corner brackets** — 90-degree inside or outside joining brackets
* **End caps** — plastic or aluminum caps for profile ends
* **Sliding nuts** — drop-in T-nuts for post-assembly insertion
* **Connecting plates** — flat plates for joining profiles end-to-end

## Usage, tips and tricks

### Measuring and cutting

* Measure twice before cutting — aluminum extrusion cut errors cannot be undone.
* Use a miter saw with a non-ferrous metal blade or a manual hacksaw with a guide for straight cuts.
* Deburr cut ends with a file or deburring tool before assembly.

### Assembling joints

* **Square (inside corner):** Use inside corner brackets with T-nuts in adjacent slots.
* **L-bracket (outside corner):** Use outside flat brackets for lighter assemblies.
* **End-to-end:** Use internal joining plates or threaded rod inserts.
* Tighten gradually — over-tightening can crack the T-nut or strip the thread.

### Load-bearing considerations

* Longer unsupported spans deflect more — add cross-bracing or choose a larger profile.
* For dynamic loads (CNC cutting forces), use heavier profiles (40x40 or 45x45) and gussets.
* Calculate maximum deflection for the profile size and load before building critical structures.

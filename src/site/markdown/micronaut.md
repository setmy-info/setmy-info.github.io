# Micronaut

## Generating mn 1.x project

For Gradle:

```sh
mn create-app micronaut-start-project
```

For Maven:

```sh
mn create-app micronaut-start-project --build maven
```

## Maven execution

Micronaut 2.x:

```sh
mvn mn:run
```

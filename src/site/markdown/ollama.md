# Ollama

## Information

`Ollama` is a local LLM runtime for downloading, serving, and running language models on your own machine.

It is one of the simplest ways to start using local AI models on `Linux`, `Windows`, and `macOS`.

### What is it for?

Typical `Ollama` use cases include:

* running open models locally with a simple install flow
* prototyping local AI applications and RAG solutions
* exposing a local inference API for desktop or backend tools
* testing prompts and models without sending data to a cloud provider
* powering local developer tools such as coding assistants and chat UIs

## Usage

`Ollama` is a strong choice when teams or individuals want the easiest local setup for experimenting with LLMs. It is
commonly paired with tools such as `Open WebUI`, `LM Studio`, vector databases, or local application frameworks.

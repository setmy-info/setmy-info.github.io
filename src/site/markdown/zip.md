## Usage, tips and tricks

zip archive.zip file1.txt file2.txt
zip -r folder.zip myfolder
zip -r folder.zip myfolder -x '*.log'
unzip -l archive.zip
unzip archive.zip
unzip archive.zip -d /path/to/directory
zip archive.zip file3.txt
zip -u archive.zip file1.txt
zip -e archive.zip file1.txt file2.txt

Action	Command
Create a Zip Archive	zip archive.zip file1.txt file2.txt
Create Zip from Directory	zip -r folder.zip myfolder
Exclude Specific Files	zip -r folder.zip myfolder -x '*.log'
List Contents	unzip -l archive.zip
Extract to Current Directory	unzip archive.zip
Extract to Specific Directory	unzip archive.zip -d /path/to/directory
Add Files to Existing Archive	zip archive.zip file3.txt
Replace Existing Files	zip -u archive.zip file1.txt
Create Password-Protected Archive	zip -e archive.zip file1.txt file2.txt

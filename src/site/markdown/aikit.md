# AIKit

## Information

`AIKit` is a Docker-based platform that uses `Unsloth` as a fine-tuning backend. It provides an `OpenAI`-compatible
interface, which makes it appealing for users who prefer container-driven workflows and familiar API integration
patterns.

## Common use cases

* container-based fine-tuning environments,
* teams that want an `OpenAI`-style API surface,
* local or self-hosted experimentation with Docker-centric workflows,
* and users who want a packaged experience instead of assembling every training component manually.

## Practical note

`AIKit` is especially useful if your preferred workflow starts with containers and service interfaces rather than direct
Python training scripts.

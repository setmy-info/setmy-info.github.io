# NemoClaw

## Information

`NemoClaw` is an open-source reference stack from `NVIDIA` for running always-on AI agents more safely inside
`OpenShell` sandboxes.

It combines sandboxing, guided onboarding, routed inference, network policy, and lifecycle management behind a single
CLI.

### What is it for?

Typical `NemoClaw` use cases include:

* running sandboxed AI agents with stronger security controls
* managing agent environments and sandbox lifecycle from a unified CLI
* applying network policies and hardening rules to agent execution
* routing inference to supported model providers
* creating a safer reference architecture for long-running agent systems

## Usage

`NemoClaw` is positioned as a reference stack for sandboxed agent execution. It works with supported agents such as
`OpenClaw`, `Hermes`, and `LangChain Deep Agents Code`, and is especially relevant when teams need stronger isolation
and operational controls for agent workloads.

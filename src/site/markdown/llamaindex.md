# LlamaIndex

## Information

`LlamaIndex` is a framework focused on data ingestion, indexing, retrieval, and building RAG-style applications on top
of LLMs.

It is especially useful when an AI application must work with documents, knowledge bases, APIs, or other private data
sources.

### What is it for?

Typical `LlamaIndex` use cases include:

* document question answering
* retrieval-augmented generation (RAG)
* indexing data from files, databases, and external systems
* building chat over internal knowledge bases
* creating ingestion pipelines for embeddings and metadata

## Usage

`LlamaIndex` is often chosen for data-centric AI applications where retrieval quality and ingestion pipelines are a core
part of the solution.

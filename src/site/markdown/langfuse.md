# Langfuse

## Information

`Langfuse` is an observability and evaluation platform for LLM applications.

It is commonly used to trace prompts, model calls, tool usage, latency, costs, and user interactions across AI
features.

### What is it for?

Typical `Langfuse` use cases include:

* tracing LLM and agent executions
* prompt management and versioning
* monitoring token usage, latency, and cost
* collecting user feedback and evaluation data
* debugging RAG and multi-step AI workflows

## Usage

`Langfuse` is often added alongside frameworks such as `LangChain`, `LangGraph`, `LlamaIndex`, `Spring AI`, or custom
provider SDK integrations when teams need better visibility into production AI behavior.
